   Data Format Returned by LORA

The LORA REST API calls all return data in JSON format. A meta-wrapper, or envelope, data structure is used in all cases. It looks like: 
  {
    output: {some-arbitrary JSON object, depending on the call},
    error:  "an error string on error, else an empty string",
    status: "OK" or "ERROR"
  }
